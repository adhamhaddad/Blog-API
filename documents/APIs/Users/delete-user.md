# Routes

## DELETE /users/{`id`}

Delete user.

### Request Headers

The request headers should have the following properties:

```json
"headers": {
    "access_token": "0c6e070d-07b8-4c38-b9db-8822e63566be"
}
```

### Response

If the request is success, the server will respond with a status code of 204.
